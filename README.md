PrePa 🎓

PrePa is a modern Computer-Based Testing (CBT) web application designed to help students prepare for JAMB and other CBT examinations.

Features
- Multiple choice CBT system
- Timer-based examination
- Question navigation
- Instant result calculation
- Responsive interface
- Built with modern UI

Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap
- Inter & Poppins fonts
